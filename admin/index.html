FLAVV ADMIN PANEL

Panoul de administrare pentru site-ul oficial

Încarcă Poze

Pentru a adăuga poze în galerie:

  1. Intri pe repository pe GitHub
  2. Accesezi folderul uploads/
  3. Butonul "Add file" → "Upload files"
  4. Tragi pozele și faci Commit
Mergi la Uploads

Modifică Text

Pentru a schimba textul site-ului:

  1. Deschizi fișierul index.html
  2. Butonul ✏️ Edit (dreapta sus)
  3. Schimbi ce text vrei
  4. Commit changes
Editează index.html

Schimbă Culori

Pentru a modifica culorile:

  1. Deschizi style.css
  2. Cauți secțiunea :root
  3. Schimbi valorile culorilor
  4. Salvezi și commit
:root { --turquoise: #00f5d4; ← Schimbi aici --turquoise-dark: #00b8a9; }
Modifică CSS
IMPORTANT:

După fiecare modificare pe GitHub, site-ul se actualizează automat în 30-60 de secunde pe Vercel.

Decap CMS (Avansat)

Pentru un panou de admin mai avansat, completează configurația de mai jos și apoi accesează isflavv.vercel.app/admin

<script> CMS.init({ config: { backend: { name: "github", repo: "stefimihai27/isFlavv", // Schimbă cu username-ul tău branch: "main", }, media_folder: "uploads", public_folder: "/uploads", collections: [ { name: "gallery", label: "Galerie Foto", folder: "uploads/gallery", create: true, fields: [ { label: "Titlu", name: "title", widget: "string" }, { label: "Poza", name: "image", widget: "image" }, { label: "Descriere", name: "description", widget: "text" } ] } ] } }); </script>

Pentru a activa Decap CMS, dezarcolează secțiunea de script de mai sus și copiaz-o la sfârșitul acestui fișier, înainte de </body>.

Vercel Dashboard Vezi Site-ul Live Înapoi la Site
